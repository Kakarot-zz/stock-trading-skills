# 龙虎榜量化席位过滤实战指南

## 概述

本指南用于过滤龙虎榜中的量化席位，识别程序化交易痕迹，保留有格局的机构和知名游资主导的票。

**核心原则**：拉萨席位 ≥ 2 → 直接排除

---

## 第一层：席位名字识别（准确率 80%+）

拉萨天团 = 东方财富互联网开户量化席位群。同一只股票出现 ≥2 个拉萨席位 = 量化确认。

### 常见拉萨席位名称

| 席位名 | 说明 |
|--------|------|
| 东方财富拉萨团结路第一 | 拉萨天团核心席位 |
| 东方财富拉萨团结路第二 | 拉萨天团核心席位 |
| 东方财富拉萨金融城南环路 | 拉萨天团核心席位 |
| 东方财富拉萨东环路第一 | 拉萨天团核心席位 |
| 东方财富拉萨东环路第二 | 拉萨天团核心席位 |
| 东方财富拉萨山南香曲东路 | 拉萨天团核心席位 |
| 东方财富拉萨娘猜曲镇路 | 拉萨天团核心席位 |
| 东方财富拉萨当雄县虎空路 | 拉萨天团核心席位 |

### 规则

```
龙虎榜上拉萨席位 ≥ 2 个 → 量化，直接排除
龙虎榜上拉萨席位 = 1 个 → 可疑，观察其他维度
龙虎榜上拉萨席位 = 0 个 → 正常，继续第二层筛选
```

---

## 第二层：行为模式识别（准确率 70%）

### 一日游
今天买明天卖。对比连续2天龙虎榜数据，如果席位完全换了一批 = 一日游量化。

### 多席位协同
3-5个陌生营业部同一天买入，且席位名称无规律（不是章盟主、溧阳路等知名游资）。

### 金额均匀
买1 = 买2 = 买3 ≈ 相同金额（分仓策略，量化均匀分配下单）。

## 第三层：DDX数据辅助

| DDX 形态 | 含义 |
|----------|------|
| DDX5 大幅为负 + 股价连板 | 量化出货 |
| DDX 持续微正微负 | 量化对倒（左手倒右手） |
| DDX 突然大幅为正 + 放量 | 量化进场信号（可关注） |

---

## 第四层：分时图形态（盘中识别）

需要实时行情数据：

| 形态 | 特征 | 含义 |
|------|------|------|
| 锯齿形 | 1分钟内频繁上下 0.1-0.3% | 程序化高频交易 |
| 精准挂单 | 整数位密集挂单 | 量化挂单策略 |
| 秒级封板 | 14:57:58 封板或 14:59:59 炸板 | 量化精准控时 |

---

## 实战口诀

```
拉萨席位 ≥ 2 个 → 量化，直接排除
同日多陌生营业部 → 可能量化分仓，警惕
DDX 连负 + 股价连板 → 量化出货，不碰
分时锯齿 + 秒级封板 → 量化操盘，远离
```

## 调用示例

```python
import os, sys, json
sys.path.insert(0, 'C:/Users/June/AppData/Local/hermes/skills/dfcf/mx-data')
with open('C:/Users/June/AppData/Local/hermes/.env') as f:
    for line in f:
        if 'MX_APIKEY' in line and not line.startswith('#'):
            os.environ['MX_APIKEY'] = line.s...'=', 1)[1].strip()
            break
import mx_data
client = mx_data.MXData()

lasa_kw = ['拉萨团结路','拉萨金融城','拉萨东环路','拉萨山南','拉萨娘猜','拉萨当雄']
r = client.query('华锋股份 今日龙虎榜席位')
tables = r.get('data',{}).get('data',{}).get('searchDataResultDTO',{}).get('dataTableDTOList',[])
for t in tables:
    for idx, row in t.get('table', {}).items():
        if idx == 'headName' or not isinstance(row, list): continue
        raw_str = json.dumps(row, ensure_ascii=False)
        lasa_hit = sum(raw_str.count(kw) for kw in lasa_kw)
        print(f"席位:{row[0]} 买额:{row[1]}万 {'⚠️量化' if lasa_hit else '✅'}")
```
