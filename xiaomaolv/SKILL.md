# 小毛驴选股体系 v3.3

> **本质：** 找有资金关注的成长股，在启动初期介入
>
> **核心公式：**
> ```
> 技术面多头排列 + 基本面增长 + 筹码集中 + 上市时间够长
> ```

## 选股范围

- **主板优先**：上证A股（600/601/000/001开头）
- **排除**：创业板（300）、科创板（688）
- 流通市值：10亿以上（不设上限）
- 上市时间：200天以上

## 五、综合筛选流程

```
第一步（技术面过滤）：
  ✅ MACD/KDJ/RSI多头发排列
  ✅ 均线多头或股价站上20日均线
  ✅ 上涨放量、下跌缩量

第二步（基本面过滤）：
  ✅ ROE > 0（必须盈利）
  ✅ 净利润增长率 > 0（业绩增长）
  ✅ 营收增长率 > 0（业务扩张）
  ✅ PEG < 1（成长性价比）

第三步（筹码面过滤）：
  ✅ 股东人数减少（筹码集中）
  ✅ 主力持仓增加（资金认可）
  ✅ 户均持股市值增加（大户进场）

第四步（参数硬条件）：
  ✅ 主板股票（600/601/000/001）
  ❌ 排除创业板（300）、科创板（688）
  ✅ 流通市值10亿以上
  ✅ 上市时间 > 200天
  ✅ 换手率3-15%
  ✅ 量比 > 1

一票否决：
  ❌ 市值 > 1000亿
  ❌ ROE < 0（亏损）
  ❌ 跌停或涨幅 > 10%
  ❌ 主力大幅流出（DDX < -0.5）
```

## 六、实战总结规律（15条核心法则）

### 规律1：多空力量决定价格走势
- **多头主导**：多方力量 > 空方力量 → 价格上涨
- **空头主导**：空方力量 > 多方力量 → 价格上涨
- **多空平衡**：双方力量相当 → 价格横盘
- **力量转换**：力量对比逆转时 → 新趋势将出现

### 规律2：趋势判断看量和价
- **温和放量**：量能温和放大 → 趋势延续信号
- **巨量见顶**：成交量暴增 → 可能是顶部信号
- **缩量见底**：成交量萎缩触底 → 可能见底
- **堆量突破**：成交量逐步放大并突破压力位 → 强势信号
- **二次放量**：回调后再创新高 → 趋势延续

### 规律3：主力行为看筹码
- **主力建仓**：低位吸筹，筹码集中
- **主力洗盘**：拉升前清洗浮筹，震仓
- **主力拉升**：脱离成本区，快速上涨
- **主力出货**：高位派发，筹码分散
- **主力被套**：高位接盘被套，可能自救

### 规律4：空间判断用黄金
- **调整支撑**：回调0.382处获支撑 → 强势
- **反弹目标**：反弹至0.618位 → 弱势
- **突破确认**：突破0.5位后，目标1.0

### 规律5：K线语言看形态
- **底部形态**：头肩底、双底、圆底
- **顶部形态**：头肩顶、双顶、圆顶
- **中继形态**：旗形、三角形、楔形
- **突破形态**：跳空缺口、突破缺口
- **持续形态**：红三兵、三阳开泰

### 规律6：技术指标共振
- **MACD+KDJ共振**：两个指标同时发出信号 → 信号更可靠
- **KDJ+RSI共振**：短线共振，灵敏度更高
- **量价+MACD共振**：量价与指标共振 → 趋势更可靠
- **MA+MACD共振**：均线与指标共振 → 趋势确认

### 规律7：背离判断转折
- **底背离**：价格新低，指标未新低 → 可能见底回升
- **顶背离**：价格新高，指标未新高 → 可能见顶回落
- **背离次数**：背离次数越多 → 转折信号越可靠
- **背离时效**：背离后需要放 量确认

### 规律8：时间周期有规律
- **斐波那契周期**：1、2、3、5、8、13、21...日容易变盘
- **均线周期**：5、10、20、60、120日均线有支撑/压力
- **短期变盘**：3-5日内会有方向选择
- **中期周期**：21日左右可能发生趋势变化
- **长期周期**：55日、144日均线是大周期转折点

### 规律9：板块联动有先后
- **龙头先动**：龙头股先于板块启动
- **跟风跟动**：其他股票跟随龙头涨跌
- **板块轮动**：热点在不同板块间转移
- **切换信号**：龙头调整时，板块可能走弱
- **资金流向**：资金从涨幅大的板块流向涨幅小的

### 规律10：消息催化加速
- **政策利好**：国家政策支持，行业迎发展机遇
- **业绩超预期**：业绩增长超预期，股价有支撑
- **产品涨价**：产品价格上涨，企业盈利预期提升
- **行业事件**：行业重大利好或利空，影响整个板块

### 规律11：仓位管理是生命线
- **顺势加仓**：趋势确认后，逐步加仓
- **逆向加仓**：摊平成本，但需严格止损
- **仓位控制**：不超过总资金的60%
- **止损原则**：单只股票亏损不超过8%
- **分批进出**：分批买入，降低择时风险

### 规律12：选股标准记心中
- **基本面**：业绩增长、ROE>0、净利润增长>0
- **技术面**：MACD/KDJ/RSI多头排列
- **资金面**：DDX>0、主力净流入
- **趋势面**：均线多头、上升通道
- **位置**：低位启动、突破关键阻力位

### 规律13：大盘环境定仓位
- **牛市**：满仓顺势，精选龙头
- **熊市**：轻仓或空仓，不逆势抄底
- **震荡市**：3-5成仓，高抛低吸
- **变盘期**：等待确认，减少操作
- **仓位调节**：根据大盘位置动态调整

### 规律14：执行纪律要严格
- **计划交易**：提前制定交易计划
- **系统执行**：按系统信号执行，不主观臆断
- **止损果断**：触及止损，无条件出局
- **止盈理性**：不贪心，到目标位分批减仓
- **定期复盘**：总结操作，不断优化系统

### 规律15：情绪管理很重要
- **不恐慌**：严格止损，防止更大亏损
- **不贪心**：到达目标位，理性止盈
- **不侥幸**：尊重市场，不逆势加仓摊平成本
- **耐心等待**：等待确定的机会再出手
- **知行合一**：系统信号与实际操作保持一致

## 九、徐总交易体系精华（实战语录）

> 以下为徐总（小马同桌）多年实战经验总结，与小毛驴体系高度互补

### 9.1 判断牛熊的三依据

| 依据 | 内容 |
|------|------|
| **大盘** | 大盘指数在所有均线之上 → 牛市；反之 → 熊市 |
| **板块** | 有持续热点板块 → 牛市；无热点 → 熊市 |
| **持仓** | 持仓股不断创新高 → 牛市；反之 → 熊市 |

### 9.2 趋势的理解与操作

- **顺势而为**：永远不与趋势作对，不在下跌时买入
- **趋势定义**：高点不断降低 = 下跌趋势；低点不断抬高 = 上涨趋势
- **右侧交易**：等趋势确认后再操作，只做确定性机会
- **左侧交易**：可在支撑位试探性买入，但需严格止损

### 9.3 三种买点

| 买点 | 特征 | 风险 |
|------|------|------|
| **顺N型回调买点** ⭐⭐⭐ | 回调结束后再次放量上涨 | 低风险 |
| **顺势中继买点** ⭐⭐ | 上涨中继整理后再启动 | 中风险 |
| **趋势尾端买点** ⭐ | 上涨末期追涨 | 高风险，不推荐 |

> **顺N型**：一涨回调一涨回调一涨，形态像字母N，是最标准的买入形态

### 9.4 量价分析口诀

```
阳线放量 → 正常上涨
阴线缩量 → 正常回调
阳线缩量 → 主力锁仓，可能加速
阴线放量 → 恐慌抛售，可能见底
堆量突破 → 主力拉升信号
二次放量 → 行情延续
```

### 9.5 看盘三层次

```
第一层：看大盘（大盘指数、涨跌家数、成交量）
第二层：看板块（热点板块、龙头板块）
第三层：看个股（持仓股、目标股）
```

### 9.6 板块轮动规律

- **规律**：资金从一个热点板块，流向另一个热点板块
- **信号**：当一个板块持续拉升 → 资金开始向低位板块转移
- **龙头切换**：旧龙头调整时，新龙头开始表现
- **出货模式**：高位横盘 → 反复拉升吸引跟风 → 大幅回落

### 9.7 热点炒作周期

| 阶段 | 特征 | 操作 |
|------|------|------|
| **启动期** | 消息刺激，板块集体拉升 | 轻仓试探 |
| **发酵期** | 龙头确立，跟风上涨 | 加仓持有 |
| **高潮期** | 板块普涨，低位股补涨 | 逐步减仓 |
| **退潮期** | 龙头破位，板块分化 | 清仓离场 |

### 9.8 龙头股操作口诀

```
擒贼先擒王，买股买龙头
龙头回调不破支撑 → 买入
龙头跌破5日线 → 减仓
龙头跌破10日线 → 清仓
```

### 9.9 判断大盘顶底的信号

**大盘阶段性顶部信号：**
- 成交量萎缩，但指数创新高
- 涨跌家数对比失调（跌多涨少）
- 热点龙头纷纷破位
- 跌停股数量增多

**大盘阶段性底部信号：**
- 成交量萎缩，指数不再创新低
- 涨跌家数对比改善
- 强势股开始抗跌
- 涨停股数量增加

### 9.10 复盘要点

```
1. 今日大盘环境如何？（趋势、成交量、热点）
2. 持仓股走势是否符合预期？
3. 目标股是否有买入机会？
4. 明日操作计划是什么？
5. 总结今日交易得失
```

### 9.11 止损方法

| 方法 | 说明 |
|------|------|
| **固定止损** | 亏损8%无条件止损 |
| **趋势止损** | 股价跌破重要均线（20日线）止损 |
| **支撑止损** | 跌破关键支撑位止损 |

> **原则**：止损要快，犹豫就会亏损扩大

### 9.12 交易系统

```
选股：热点板块 + 技术面多头 + 主力资金关注
买入：顺N型买点 + 突破确认 + 缩量回调支撑
持有：趋势不破不卖
卖出：趋势破坏 / 达到目标 / 基本面变化
止损：8%或趋势破坏（二者先到）
```

### 9.13 心态管理

- 不后悔：卖错的股票还会涨，遵守系统就好
- 不较劲：市场永远是对的，不与市场争辩
- 耐心：等待确定的机会，减少无效交易
- 纪律：计划你的交易，交易你的计划

### 9.14 盘感训练

```bash

# 稳健版（低换手+技术多头）

```

### 方式二：分步筛选

1. **第一步**：用 `mx-xuangu` 筛选基础条件（流通市值10亿以上、换手率3-15%）
2. **第二步**：用 `mx-data` 查询MACD状态、股东人数变化、主力持仓
3. **第三步**：核实财务数据（ROE > 0、净利润增长 > 0、营收增长 > 0）
4. **第四步**：确认量价配合（上涨放量、下跌缩量）

### 方式三：南瓜鱼+小毛驴双体系综合评分（推荐）

当市场无三连板标的时，用 `references/comprehensive-scoring.md` 的评分系统对候选股打分：

```
综合得分 = 南瓜鱼得分 × 0.6 + 小毛驴得分 × 0.4
```

**标准流程**：
1. 用妙想选股API全市场扫描「近10日有涨停 涨停次数>=2 主板 市值10-80亿 PE>0」
2. 过滤ST/非主板/PE>200后，对候选股查K线+财务数据
3. 用综合评分系统打分，选TOP2
4. **所有数据必须用妙想API**，不能用Sina K线做投资决策
