# 小毛驴选股体系 v3.3

> **本质：** 找有资金关注的成长股，在启动初期介入
>
> **核心公式：**
> ```
> 技术面多头排列 + 基本面增长 + 筹码集中 + 上市时间够长
> ```

## 选股范围

- **主板优先**：上证A股（600/601/000/001开头）
- **排除**：创业板（300）、科创板（688）
- 流通市值：10亿以上（不设上限）
- 上市时间：200天以上

## 五、综合筛选流程

```
第一步（技术面过滤）：
  ✅ MACD/KDJ/RSI多头发排列
  ✅ 均线多头或股价站上20日均线
  ✅ 上涨放量、下跌缩量

第二步（基本面过滤）：
  ✅ ROE > 0（必须盈利）
  ✅ 净利润增长率 > 0（业绩增长）
  ✅ 营收增长率 > 0（业务扩张）
  ✅ PEG < 1（成长性价比）

第三步（筹码面过滤）：
  ✅ 股东人数减少（筹码集中）
  ✅ 主力持仓增加（资金认可）
  ✅ 户均持股市值增加（大户进场）

第四步（参数硬条件）：
  ✅ 主板股票（600/601/000/001）
  ❌ 排除创业板（300）、科创板（688）
  ✅ 流通市值10亿以上
  ✅ 上市时间 > 200天
  ✅ 换手率3-15%
  ✅ 量比 > 1

一票否决：
  ❌ 市值 > 1000亿
  ❌ ROE < 0（亏损）
  ❌ 跌停或涨幅 > 10%
  ❌ 主力大幅流出（DDX < -0.5）
```

## 六、实战总结规律（15条核心法则）

### 规律1：多空力量决定价格走势
- **多头主导**：多方力量 > 空方力量 → 价格上涨
- **空头主导**：空方力量 > 多方力量 → 价格上涨
- **多空平衡**：双方力量相当 → 价格横盘
- **力量转换**：力量对比逆转时 → 新趋势将出现

### 规律2：趋势判断看量和价
- **温和放量**：量能温和放大 → 趋势延续信号
- **巨量见顶**：成交量暴增 → 可能是顶部信号
- **缩量见底**：成交量萎缩触底 → 可能见底
- **堆量突破**：成交量逐步放大并突破压力位 → 强势信号
- **二次放量**：回调后再创新高 → 趋势延续

### 规律3：主力行为看筹码
- **主力建仓**：低位吸筹，筹码集中
- **主力洗盘**：拉升前清洗浮筹，震仓
- **主力拉升**：脱离成本区，快速上涨
- **主力出货**：高位派发，筹码分散
- **主力被套**：高位接盘被套，可能自救

### 规律4：空间判断用黄金
- **调整支撑**：回调0.382处获支撑 → 强势
- **反弹目标**：反弹至0.618位 → 弱势
- **突破确认**：突破0.5位后，目标1.0

### 规律5：K线语言看形态
- **底部形态**：头肩底、双底、圆底
- **顶部形态**：头肩顶、双顶、圆顶
- **中继形态**：旗形、三角形、楔形
- **突破形态**：跳空缺口、突破缺口
- **持续形态**：红三兵、三阳开泰

### 规律6：技术指标共振
- **MACD+KDJ共振**：两个指标同时发出信号 → 信号更可靠
- **KDJ+RSI共振**：短线共振，灵敏度更高
- **量价+MACD共振**：量价与指标共振 → 趋势更可靠
- **MA+MACD共振**：均线与指标共振 → 趋势确认

### 规律7：背离判断转折
- **底背离**：价格新低，指标未新低 → 可能见底回升
- **顶背离**：价格新高，指标未新高 → 可能见顶回落
- **背离次数**：背离次数越多 → 转折信号越可靠
- **背离时效**：背离后需要放 量确认

### 规律8：时间周期有规律
- **斐波那契周期**：1、2、3、5、8、13、21...日容易变盘
- **均线周期**：5、10、20、60、120日均线有支撑/压力
- **短期变盘**：3-5日内会有方向选择
- **中期周期**：21日左右可能发生趋势变化
- **长期周期**：55日、144日均线是大周期转折点

### 规律9：板块联动有先后
- **龙头先动**：龙头股先于板块启动
- **跟风跟动**：其他股票跟随龙头涨跌
- **板块轮动**：热点在不同板块间转移
- **切换信号**：龙头调整时，板块可能走弱
- **资金流向**：资金从涨幅大的板块流向涨幅小的

### 规律10：消息催化加速
- **政策利好**：国家政策支持，行业迎发展机遇
- **业绩超预期**：业绩增长超预期，股价有支撑
- **产品涨价**：产品价格上涨，企业盈利预期提升
- **行业事件**：行业重大利好或利空，影响整个板块

### 规律11：仓位管理是生命线
- **顺势加仓**：趋势确认后，逐步加仓
- **逆向加仓**：摊平成本，但需严格止损
- **仓位控制**：不超过总资金的60%
- **止损原则**：单只股票亏损不超过8%
- **分批进出**：分批买入，降低择时风险

### 规律12：选股标准记心中
- **基本面**：业绩增长、ROE>0、净利润增长>0
- **技术面**：MACD/KDJ/RSI多头排列
- **资金面**：DDX>0、主力净流入
- **趋势面**：均线多头、上升通道
- **位置**：低位启动、突破关键阻力位

### 规律13：大盘环境定仓位
- **牛市**：满仓顺势，精选龙头
- **熊市**：轻仓或空仓，不逆势抄底
- **震荡市**：3-5成仓，高抛低吸
- **变盘期**：等待确认，减少操作
- **仓位调节**：根据大盘位置动态调整

### 规律14：执行纪律要严格
- **计划交易**：提前制定交易计划
- **系统执行**：按系统信号执行，不主观臆断
- **止损果断**：触及止损，无条件出局
- **止盈理性**：不贪心，到目标位分批减仓
- **定期复盘**：总结操作，不断优化系统

### 规律15：情绪管理很重要
- **不恐慌**：严格止损，防止更大亏损
- **不贪心**：到达目标位，理性止盈
- **不侥幸**：尊重市场，不逆势加仓摊平成本
- **耐心等待**：等待确定的机会再出手
- **知行合一**：系统信号与实际操作保持一致

## 九、徐总交易体系精华（实战语录）

> 以下为徐总（小马同桌）多年实战经验总结，与小毛驴体系高度互补

### 9.1 判断牛熊的三依据

| 依据 | 内容 |
|------|------|
| **大盘** | 大盘指数在所有均线之上 → 牛市；反之 → 熊市 |
| **板块** | 有持续热点板块 → 牛市；无热点 → 熊市 |
| **持仓** | 持仓股不断创新高 → 牛市；反之 → 熊市 |

### 9.2 趋势的理解与操作

- **顺势而为**：永远不与趋势作对，不在下跌时买入
- **趋势定义**：高点不断降低 = 下跌趋势；低点不断抬高 = 上涨趋势
- **右侧交易**：等趋势确认后再操作，只做确定性机会
- **左侧交易**：可在支撑位试探性买入，但需严格止损

### 9.3 三种买点

| 买点 | 特征 | 风险 |
|------|------|------|
| **顺N型回调买点** ⭐⭐⭐ | 回调结束后再次放量上涨 | 低风险 |
| **顺势中继买点** ⭐⭐ | 上涨中继整理后再启动 | 中风险 |
| **趋势尾端买点** ⭐ | 上涨末期追涨 | 高风险，不推荐 |

> **顺N型**：一涨回调一涨回调一涨，形态像字母N，是最标准的买入形态

### 9.4 量价分析口诀

```
阳线放量 → 正常上涨
阴线缩量 → 正常回调
阳线缩量 → 主力锁仓，可能加速
阴线放量 → 恐慌抛售，可能见底
堆量突破 → 主力拉升信号
二次放量 → 行情延续
```

### 9.5 看盘三层次

```
第一层：看大盘（大盘指数、涨跌家数、成交量）
第二层：看板块（热点板块、龙头板块）
第三层：看个股（持仓股、目标股）
```

### 9.6 板块轮动规律

- **规律**：资金从一个热点板块，流向另一个热点板块
- **信号**：当一个板块持续拉升 → 资金开始向低位板块转移
- **龙头切换**：旧龙头调整时，新龙头开始表现
- **出货模式**：高位横盘 → 反复拉升吸引跟风 → 大幅回落

### 9.7 热点炒作周期

| 阶段 | 特征 | 操作 |
|------|------|------|
| **启动期** | 消息刺激，板块集体拉升 | 轻仓试探 |
| **发酵期** | 龙头确立，跟风上涨 | 加仓持有 |
| **高潮期** | 板块普涨，低位股补涨 | 逐步减仓 |
| **退潮期** | 龙头破位，板块分化 | 清仓离场 |

### 9.8 龙头股操作口诀

```
擒贼先擒王，买股买龙头
龙头回调不破支撑 → 买入
龙头跌破5日线 → 减仓
龙头跌破10日线 → 清仓
```

### 9.9 判断大盘顶底的信号

**大盘阶段性顶部信号：**
- 成交量萎缩，但指数创新高
- 涨跌家数对比失调（跌多涨少）
- 热点龙头纷纷破位
- 跌停股数量增多

**大盘阶段性底部信号：**
- 成交量萎缩，指数不再创新低
- 涨跌家数对比改善
- 强势股开始抗跌
- 涨停股数量增加

### 9.10 复盘要点

```
1. 今日大盘环境如何？（趋势、成交量、热点）
2. 持仓股走势是否符合预期？
3. 目标股是否有买入机会？
4. 明日操作计划是什么？
5. 总结今日交易得失
```

### 9.11 止损方法

| 方法 | 说明 |
|------|------|
| **固定止损** | 亏损8%无条件止损 |
| **趋势止损** | 股价跌破重要均线（20日线）止损 |
| **支撑止损** | 跌破关键支撑位止损 |

> **原则**：止损要快，犹豫就会亏损扩大

### 9.12 交易系统

```
选股：热点板块 + 技术面多头 + 主力资金关注
买入：顺N型买点 + 突破确认 + 缩量回调支撑
持有：趋势不破不卖
卖出：趋势破坏 / 达到目标 / 基本面变化
止损：8%或趋势破坏（二者先到）
```

### 9.13 心态管理

- 不后悔：卖错的股票还会涨，遵守系统就好
- 不较劲：市场永远是对的，不与市场争辩
- 耐心：等待确定的机会，减少无效交易
- 纪律：计划你的交易，交易你的计划

### 9.14 盘感训练

## 十一、实战临盘体系（5月行情提炼）

> 来源：5月复盘答疑环节提炼，2026-05实战验证

---

### 技能一：指数共振与择时技能（Market Timing Skill）

**目标**：解决"什么时候买/卖"的问题，避免盲目追高。

#### Skill 1.1：关键节点右侧决策能力

**核心逻辑**：放弃主观预测，只在指数关键节点（首次分歧、修复日）做出应对。

**动作固化**：
- 当指数处于上升趋势，**首次出现分歧放量阴线**时 → 次日尾盘寻找探底回升的抗跌品种（如力通电子案例）
- 当指数连续下跌后出现**第一个拐点日** → 执行"节点日右侧追涨"或"尾盘低吸"

**实战案例（5月教训）**：
- 指数分歧日出现放量阴线 → 次日尾盘抗跌股 > 盲目追涨当日强势股

---

#### Skill 1.2：板块轮动"事不过三"审判

**核心逻辑**：识别主线题材的生命周期。

**动作固化**：
监控主线（如芯片）。若板块经历**多次（3次及以上）回流**后，资金开始流向非核心方向（如电力）→ 立即判定主线进入衰退期 → 执行防守策略，关注抱团或投机方向。

**量化规则**：
```
主线板块经历 ≥3次 资金回流 → 衰退信号 → 切换防守
非核心方向（电力/零售）出现批量涨停 → 主线资金撤退确认
```

---

#### Skill 1.3：主线强弱鉴别术

**核心逻辑**：以5月行情为例，观察修复日表现。

**动作固化**：
在指数修复日，**最强表现的板块 = 当月主线**。若某题材在关键修复日毫无表现 → 直接判定为过渡性/配角题材，**一票否决，果断放弃**。

**实战验证（5月行情）**：
- 修复日电力板块批量涨停 → 资金确认电力为主线
- 修复日毫无表现的板块 → 后续持续弱势

---

### 技能二：选股与流动性风控技能（Stock Selection & Liquidity Skill）

**目标**：解决"买什么"以及"怎么防范闪崩"的问题。

#### Skill 2.1：龙头股识别与"大资金思维"

**核心逻辑**：倾向于"庄家"或机构主导、逻辑清晰且流动性充足的标的。

**动作固化**：
选股时必须满足四大指标：**控盘能力强、承接力强、走势稳健、大资金进出方便**。剔除无逻辑、无承接的纯资金拉抬标的。

**一票否决信号**：
```
❌ 无明确催化逻辑（无消息/无业绩/无政策）
❌ 分时图呈现量化锯齿形（1分钟内频繁上下0.1-0.3%）
❌ 精准挂单：整数位密集挂单
❌ 秒级封板/炸板：14:57:58封板或14:59:59炸板
```

---

#### Skill 2.2：劣质流动性过滤（黑名单机制）

**核心逻辑**：防范缺乏流动性的"小庄"或量化控盘票造成的闪崩。

**动作固化**：
建立选股过滤器，**日均成交额过低、走势分时图明显有量化痕迹的标的，一律不参与**，防止大资金无法顺利离场造成巨大回撤。

**量化硬条件**：
```
❌ 日均成交额 < 3亿（主板个股）
❌ 拉萨席位 ≥2 个（量化分仓嫌疑）
❌ 分时图出现秒级封板/炸板（量化操盘特征）
❌ DDX 连续3日为负（主力持续出货）
```

**黑名单触发条件**：
```
单笔亏损 > 15% → 立即加入"黑名单"死牢，暂不操作
亏损后 → 全体人员（或个人）针对该股票进行全面复盘，分析亏损的共性错误
```

---

### 技能三：资金与仓位管理技能（Position Management Skill）

**目标**：解决"买多少"的问题，用仓位控制回撤。

#### Skill 3.1：确定性导向的分仓策略（非对称交易）

**核心逻辑**：提倡在**确定性高的节点重仓出击**，震荡期轻仓。

**动作固化**：

| 机会等级 | 场景 | 仓位 | 说明 |
|---------|------|------|------|
| **A级**（高确定性节点） | 四方达案例：节点日确认板块走强，次日爆发确认加仓 | 20%（满仓） | 锁定20%-30%短线利润 |
| **B级**（修复日确认） | 指数拐点后首个修复日，板块批量涨停确认 | 15% | 主线方向明确 |
| **C级**（震荡期/非节点） | 板块轮动，无明确主线 | 5%-10% | 轻仓试错 |

**严禁**：在下降趋势中盲目推高仓位。

---

#### Skill 3.2：短线持股"知行合一"

**核心逻辑**：主升段重在持有，而非频繁做T；长线看认知，而非死扛。

**动作固化**：
- 在**主升浪阶段**，保持定力，减少日内T+0操作，避免"卖飞"核心利润
- 无法理解估值逻辑的股票，一旦出现后续调整，**绝不死扛**，防止错失其他机会（中际旭创反面案例）

---

### 技能四：风控与自进化技能（Risk Control & Evolution Skill）

**目标**：解决"输了怎么办"以及"如何长期生存"的问题。

#### Skill 4.1：硬性熔断与熔断复盘机制

**核心逻辑**：强调用亏损控制回撤幅度，避免重复犯错。

**动作固化**：

| 熔断线 | 触发条件 | 处理方式 |
|--------|---------|---------|
| **8%硬止损** | 单笔亏损达到8% | 无条件止损出局 |
| **15%黑名单** | 单笔亏损超过15% | 立即加入黑名单，暂不操作 |
| **-7%硬规则** | 持仓亏损达到7% | 强制清仓（模拟账户铁律）|

**熔断复盘机制**：
```
触发15%熔断 → 个人/团队复盘 → 分析亏损共性错误 → 更新选股过滤规则
每次复盘教训 → 固化为新的排除条件 → 加入Skill 2.2黑名单机制
```

---

#### Skill 4.2：知行合一情绪控制

**核心逻辑**：克服恐慌割肉和赌气硬扛，保持常态与空杯心态。

**动作固化**：
在市场非理性暴跌时，**强制切断主观情绪**，交由上述交易系统（如触发止损线）自动执行，做到"知行合一"。

**情绪失控信号（自我检测）**：
```
⚠️ 亏损后想"赌一把"重仓追回来 → 立即停手
⚠️ 盈利后过度自信仓位失控 → 立即降回标准仓位
⚠️ 连续亏损3次后情绪化交易 → 强制休息1天
```

---

### 临盘过滤法则（尾盘低吸专项）

**核心逻辑**：次日有明确回流或加强预期 → 可低吸（具性价比）；次日预期不明 → 放弃出手。

**量化判断标准**：

| 次日预期信号 | 结论 | 操作 |
|-------------|------|------|
| 板块涨停股次日溢价高开 | 回流预期明确 | ✅ 可尾盘低吸 |
| 龙头股分时图强势整理 | 板块未退潮 | ✅ 可尾盘低吸 |
| 板块无任何催化，涨停数量少 | 预期不明 | ❌ 放弃出手 |
| 龙头股高开低走，板块跌多涨少 | 退潮确认 | ❌ 观望 |

**月底临近操作原则**：
```
✅ 只跟随盘面，有新东西（新赚钱信号）才动
❌ 绝不主观预判"月底切题材"
没信号 → 看盘不做操作
```
