## 关键教训：Sina K线接口1天延迟

**问题**：Sina K-line接口 `CN_MarketData.getKLineData` 在5/19收盘后查询，返回数据最新日期仅为5/18。
**影响**：无法用K线计算5/19收盘价RSI。
**解决**：使用 `hq_str_` 实时行情接口获取当日收盘价，K线仅作历史趋势参考。

```python
# ❌ 错误：K线只有5/18数据
kl = get_kline(sym, 20)  # 最新一根是5/18

# ✅ 正确：用hq_str获取当日实时收盘价
hq = sina_hq([("联创光电", "sh600363")])
close_today = hq["sh600363"]["close"]  # 5/19真实收盘价
```
