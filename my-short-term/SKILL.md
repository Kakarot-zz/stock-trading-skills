## 二、选股条件

### 2.1 南瓜鱼·硬性条件（连板股专用）

| 条件 | 标准 | 说明 |
|------|------|------|
| 连板数 | **三连板以上**（必须是连续交易日的连续涨停） | 短板之后进入自选，强势股必须经受过市场检验 |
| 涨停频率 | 10个交易日内 ≥ 3次涨停（含非连续） | 说明资金反复参与，股性极活 |
| 自由市值 | 10亿 < 市值 ≤ **60亿**（最优），≤ **70亿**（可接受） | 太大小盘游资拉不动，太小流动性差；超过70亿直接排除 |
| 板块 | 主流板块（板块涨幅是大盘的3倍以上） | 主流资金聚集，跟风盘多，成功率更高 |
| 市场 | 仅主板 | 排除ST、创业板（波动过大） |

### 2.2 个人体系·选股标准（非连板股）

```
短线快速拉伸标的特征：

1. 题材明确：政策/事件催化当天或次日能发酵
2. 股性活跃：成交额 > 10亿，日振幅大
3. 位置适中：回调支撑位低吸，不追高
4. 主力痕迹：放量拉升有大手笔成交
5. 板块联动：同一板块有2只以上涨停
```

### 2.3 选股公式

```
短线强势股（南瓜鱼）= 三连板+ AND (10日内涨停≥3次) AND (10亿<自由市值≤60亿) AND (主板)

【关键原则】先确认主线板块，再从中选股
- 主线板块特征：板块今日涨幅是大盘的3倍以上，有明确政策/事件催化
- 弱板块（板块涨幅<大盘或下跌）中的个股即使技术面好也优先排除
```

### 2.4 全市场扫描工作流（非涨停候选·2026-05-29确认）

> 当用户要求"从主板全市场扫描"时，使用以下工作流，不是只在涨停池里选：

```
Step 0（必做）：识别主线板块

  → Sina涨幅排行 vip.stock.finance.sina.com.cn/quotes_service/api/json_v2.php

  → 筛选条件：涨幅3%~9%（排除涨停和弱势） + 成交额5亿+
  → 腾讯qt.gtimg.cn批量获取真实涨跌幅和成交额（不用Sina的f[32]涨跌幅字段）

  → RSI-14计算：腾讯K线 web.ifzq.gtimg.cn/appstock/app/fqkline/get

## 三、买点体系

### 3.1 南瓜鱼买点（连板股回调低吸）

#### 条件一：短板日放巨量大跌

- **理想形态**：连板股在某一交易日出现放巨量大阴线（跌幅5%以上）
- **本质**：资金有分歧，原主力出货，但新资金承接积极
- **危险信号**：严重缩量跌停 → 排除A杀风险，缩量跌停是流动性枯竭，不能买
- **必要条件**：股价不能有效跌破10日线
  - 跌破10日线 = 旧主力已出货，抄底资金未到位，股价大概率继续跌
  - 10日线是强势股最后的底线

#### 条件二：筹码高度集中

- **理想形态**：70%以上筹码在1个交易日内完成集中
- **极品形态**：90%以上筹码在1个交易日内同步迅速集中
- **本质**：筹码从分散快速变为高度集中，说明新主力已控盘

#### 买点规则

```
✅ 买点 = 条件一（巨量回调未破10日线）+ 条件二（筹码集中）

买入时机：
- 条件一二同时满足 → 越低位置越买，买入时机接近尾盘越好
- 市场越差（短线情绪冰点）→ 成功率越高
- 更强势的个股：不会给10日线附近机会，在5日线附近完成条件一二的回调即启动
```

#### 买入优先级

```
第一梯队：三连板以上 + 巨量回调到10日线 + 筹码集中 → 尾盘买入
第二梯队：三连板以上 + 巨量回调到5日线 + 筹码集中 → 次日开盘确认后买
回避：缩量跌停、已跌破10日线、筹码未集中
```

### 3.2 个人买点（共振买入法·非连板股）

#### 理想买入条件（需同时满足）

```
✅ 价格条件：回调至MA20附近（±1%）企稳，或触及近期低点支撑
✅ 均线条件：MA5上穿MA10（金叉），或多头排列初现
✅ MACD条件：DIFF由负转正（水下金叉），或绿柱连续收缩
✅ 量能条件：放量阳线（成交量 > 前日1.5倍），价升量增
```

**综合买点 = 回调到位 + MACD转多 + 放量确认（三点共振才下手）**

#### 买入优先级

1. **首选**：主线板块（机器人/航天/F5G）中回调到MA5/MA10的强势股
2. **次选**：跟风补涨股（板块联动，中天强则亨通补涨）
3. **不碰**：高位出货形态（西藏珠峰式放量大跌+高换手）

### 三种买点场景

| 场景 | 时机 | 逻辑 |
|------|------|------|
| 主线分歧尾盘 | 14:30~14:50 低吸核心 | 抛压释放，次日修复 |
| 弱转强次日 | 竞价平开/小高开(<2%) | 昨日封板资金锁仓，抛压小 |
| 突发消息首板 | 竞价后半段~开盘5分钟 | 逻辑驱动第一时间 |

## 3.4 弱转强（Weak-to-Strong）系统判断框架

> 本节为2026-05-28新增。弱转强是超短线最重要的买点结构之一，必须系统掌握。

### 弱转强定义

**弱转强 = 前日走势很弱（炸板/烂板/尾盘勉强封住），但次日竞价/开盘表现异常强势**

这是主力故意制造"弱势假象"洗盘后，次日用竞价/开盘直接证明控盘力的结构。核心逻辑：昨日封板资金锁仓 → 次日抛压小 → 缩量高开/平开快速拉升。

### 弱转强三步诊断法

```
Step 1：识别前日"弱"形态（满足任一即算弱）
  ├─ 炸板（涨停打开，尾盘勉强回封）
  ├─ 烂板（反复开板超过3次）
  ├─ 尾盘板（14:30后才封板，封板质量差）
  └─ 缩量板（封板量能不足，抛压未出清）

Step 2：确认次日"强"信号（满足任一即算强）
  ├─ 竞价高开 > 1%（理想1%~3%）
  ├─ 开盘5分钟内股价创日内新高
  ├─ 成交额比昨日竞价时段放大 ≥ 30%
  └─ 分时均线快速站上（回调不破均线）

Step 3：共振验证（加分项，越多越强）
  ├─ RSI<80（安全边际）
  ├─ 板块情绪回暖（跟风盘多）
  ├─ 无量化席位（龙虎榜确认）
  └─ 距涨停 < 5%（封板动能未衰竭）
```

### 弱转强量化标准

| 指标 | 弱转强确认值 | 说明 |
|------|------------|------|
| 竞价涨幅 | +1% ~ +3% | 过高=兑现，过低=不及预期 |
| 开盘30分钟走势 | 价格 > 开盘价 + 1% | 稳定拉升，不破分时均线 |
| 成交量 | 为昨日全天的 30%~50%（竞价段） | 缩量高开 = 锁仓确认 |
| RSI | < 80（理想60~75） | RSI>80 弱转强容易失败 |
| 距涨停 | < 5% | 封板动能尚在 |
| 龙虎榜 | 无量化席位（拉萨<2） | 量化席位次日砸盘概率高 |

### 弱转强失败模式

| 形态 | 特征 | 操作 |
|------|------|------|
| 高开低走 | 竞价高开>3% → 开盘后单边下跌 | 立刻止损 |
| 冲高回落 | 开盘拉升 → 10:00前见顶 → 逐步回落 | 止盈或止损 |
| 缩量平开低走 | 竞价平开 → 缩量阴跌 | 止损 |
| RSI过热+弱转强 | RSI>85 + 高开 | 弱转强结构被RSI过热抵消，失败率高 |

### 弱转强 vs 强转强（重要区分）

```
弱转强：前日弱势（炸板/烂板）→ 次日竞价超预期 → 买点：竞价/开盘5分钟
强转强：前日强势（缩量一字板）→ 次日高开 → 买点：等回调MA5，不追高
       （强转强追高容易买在日内最高）
```

### 弱转强速记口诀

```
弱转强三看：
  昨日够不够弱（炸板/烂板/尾盘板）？
  今日够不够强（竞价高开1~3%，缩量）？
  RSI够不够低（<80才安全，>85禁止买）？

弱转强三不看：
  RSI>80，不看（过热区弱转强容易失败）
  量化席位≥2，不看（次日必砸）
  板块弱势，不看（跟风盘不足）
```

## 四、RSI梯度法则（连板股专用）

> 2026-05-21 实盘教训整合：连板股的RSI必须分类讨论，普通RSI规则不适用

### RSI梯度决策表（连板股回调场景）

| RSI区间 | 操作 |
|---------|------|
| RSI < 40 | ✅ 强烈关注，等回调到位买 |
| RSI 40-60 | ✅ 正常关注，买点较好 |
| RSI 60-75 | ⚠️ 可接受，注意控制仓位（≤10%） |
| RSI 75-80 | ⚠️ 接近超买，高开不追，等次日回踩MA5 |
| **RSI > 80** | ❌ 排除，不买（可加入自选等RSI<70再评估） |
| **RSI > 95** | ❌ 无条件排除（连板终极规则） |
| **RSI = 100（涨停中）** | ✅ 正常，不作为排除依据 | 涨停导致的数值溢出，不是真实RSI超买 |

### RSI<20 不等于"超卖可以买"（连板股分类讨论）

| 场景 | RSI含义 | 操作 |
|------|---------|------|
| 普通股超跌反弹 | RSI<30=超卖，反弹概率高 | ✅ 可以关注买点 |
| 连续涨停股（连板中）| RSI<30=连板消耗了全部卖盘，RSI低是强势标志 | ❌ 不是买点，是持筹者盛宴 |
| 连板后回调（尚未跌完）| RSI<30=卖压未出清，可能继续跌 | ❌ RSI低可以更低（通达股份RSI=28照样跌）|

### RSI从高位快速回落 = 主力洗盘

```
✅ 可以买：连板过程中 RSI 从 70+ 快速回落至 50-60（主力洗盘）
❌ 不能买：连板过程中 RSI 一直在 20-40 低位（根本没有像样洗盘，RSI 低是假的）
❌ 不能买：连板后 RSI<30 但价格尚未回调（还没到买点）
```

### RSI-6 超买预警（短线持仓1-5日）

- RSI-6 < 50：✅ 中性偏低，可买
- RSI-6 50-70：✅ 正常上涨区间
- RSI-6 70-80：⚠️ 偏热，短线持仓可以买但需控仓
- **RSI-6 > 80：❌ 极度超买，回撤概率超过70%**
- RSI-6 从低位（如40）快速拉升到80+：⚠️ 主力拉升末端标志，1-3日内必回撤

## 五、卖出信号（止盈止损矩阵）

### 5.1 止损规则（满足即出）

| 触发条件 | 操作 |
|----------|------|

| **亏损达到 7%** | **无条件止损（硬规则，不可更改）** |

**核心理念：短线不补仓，止损要坚决。补仓是散户死亡陷阱。**

### 5.2 止盈规则

| 涨幅 | 操作 |
|------|------|
| 涨 > 5% | 止盈一半，锁住利润 |
| 涨 > 8% | 再止盈25% |
| 涨 > 12% | 剩余移动止盈（MA5为止盈线） |

### 5.3 卖出信号（满足任一即出）

```
❌ 均线偏离：价格偏离MA5 > 5%（乖离过大）
❌ MACD死叉：DIFF由正转负
❌ 量价背离：价格创新高但成交量萎缩
❌ 均线压制：MA5下穿MA10（死叉）+ 价格跌破MA10
❌ 高位出货形态：换手率 > 10% + 放量大跌 + 高位
```

### 5.4 连板股专用卖出规则

#### 炸板卖出

| 情形 | 操作 |
|------|------|
| 炸板后不回封 | 清仓 |
| 炸板后回封 | 减半仓，剩余移动止盈 |
| 炸板幅度 > 5% | 无条件清仓 |

#### 低开卖出

| 情形 | 操作 |
|------|------|
| 大低开（>-3%） | 可以格局，等待上冲机会（开盘后观察30分钟），若上冲无力则止损 |
| 小低开（≤-3%） | 观察同板块核心股走势，若跟风走弱则减仓 |

#### 二连板后高开规则（宁波中百教训·2026-05-15）

```
❌ 旧规则：二连板后高开 → 等回调MA5买（错误）
✅ 新规则：
  1. 二连板 + RSI>85 → 当日涨停应减仓，不追
  2. 二连板次日高开（>3%）→ 立即止盈，不买入
  3. 高开后回落跌破分时均线30分钟 → 清仓
  4. 二连板后只有"低开/平开+缩量回调到MA10"才是买点

## 七、买入判断四维度框架

> 复盘后次日买入决策的核心过滤器。四个维度同时满足方可买入。

### 四维买入规则

| 维度 | 判断标准 | 不满足则 |
|------|---------|---------|
| **板块** | 板块成交额排名前三 | 不买（情绪孤岛） |
| **资金** | 龙虎榜无量化席位（拉萨系≥2个勿碰） | 不买（次日砸盘） |
| **催化** | 有实质催化：业绩/招股书更新/政策/供应链公告 | 不买（纯炒作） |
| **价格** | 竞价金额比昨日更大 + RSI<85 + MA5乖离<10% | 管住手 |

### 六种禁止买入

| 情况 | 原因 |
|------|------|
| 高开>3%买 | 溢价过高，次日低开概率大 |
| 尾盘涨停后抢筹 | 无逻辑支撑，次日大概率低开 |
| 量化席位≥2 | 次日砸盘，程序化出货 |
| 人气榜吹票满天飞 | 短期顶部信号 |
| RSI>85极度超买 | 回调压力大，赔率差 |
| 板块高潮次日早盘追 | 卖点不是买点 |

### 决策流程

```
今日最强板块成交额前三？
  ├─ 否 → 不买
  └─ 是 → 板块内个股满足？
            ├─ 龙虎榜无量化席位？
            │   ├─ 否 → 不买
            │   └─ 是 → 有实质催化？
            │           ├─ 否 → 不买
            │           └─ 是 → 竞价金额比昨日更大？
            │                   ├─ 否 → 管住手
            │                   └─ 是 → RSI<85 + MA5乖离<10%？
            │                           ├─ 否 → 等回调
            │                           └─ 是 → 买点确认，买入
```

## 八、持仓 exit decision checklist

> 用法：每日盘中或收盘后，对持仓逐只过筛，决定留/走/加。
> 数据获取：当前价→mx-data API（东方财富）；持仓成本→妙想 mx-moni positions 接口

### 8.1 持仓个股 exit 矩阵

| 信号类型 | 阈值 | 操作 |
|----------|------|------|
| 跌破成本价 3% | 收盘价 < 成本 × 0.97 | 无条件止损 |
| 跌破 20 日线（收盘确认） | 价格 < MA20 × 0.98 | 止损或收盘前清 |
| 均线死叉 + 放量 | MA5 下穿 MA10 + 量 > 5 日均量 1.5 倍 | 减仓 50% |
| 大盘涨它不涨（跑输指数 3%+） | 个股 pct < 指数 pct - 3 | 换仓或止损 |
| 盈利回吐超过 50% | (现价-成本) 浮盈回撤 > 50% | 止盈线收紧至成本价 |
| 放量破位（跌幅 > 3% 伴大量） | 跌幅 > 3% + 量比 > 2 | 立刻止损 |

### 8.2 止损线执行：真破位 vs 主力故意砸盘

> 2026-05-21 实战教训：止损线触发 ≠ 立刻市价砸盘

**止损线执行决策矩阵**

| 破线后走势 | 特征 | 操作 |
|-----------|------|------|
| 放量急跌破线 → 瞬间拉回 | 主力故意砸止损盘 | 观察，不卖 |
| 缩量阴跌破线 → 反弹无力 | 真破位 | 立刻执行止损 |
| 盘中瞬间刺破（<5分钟）立刻站回 | 假破位 | 不卖，等收盘确认 |
| 收盘价跌破止损线 | 确认破位 | 次日开盘就走 |
| 低开后缩量横盘 + 大单护盘 | 主力在接筹 | 不卖，等主力拉升 |

**量化砸盘 vs 真破位辨认**

```
量化砸盘特征：
  - 砸得凶：单笔大单砸穿止损线
  - 拉得快：5-15分钟内收复
  - 成交回报：卖出席位多为量化席位（拉萨系）

真破位特征：
  - 跌下去缩量（无人接盘）
  - 反弹无力（每次反弹高点降低）
  - 成交回报：卖出席位分散，无量化特征
```

**实操口诀**
```
止损线预警价：距离止损线3%以内设预警
到达预警 → 盯盘，不挂市价单
确认真破 → 市价止损
疑似假破 → 等收盘，收盘确认破再走
```

### 8.3 持仓分类处理

```
【强势持仓】pct ≥ 3% & 量增 & 均线多头
  → 持有，移动止盈 MA5

【正常持仓】pct 在 ±2% 之间
  → 持有，观察明日开盘方向

【弱势持仓】pct ≤ -2% & 跑输大盘
  → 止损或收盘前出

【周五新开仓（最危险！）】
  → 次日若跑输大盘 1%+，立刻检查：
     1. 20 日线是否破（关键支撑）
     2. 量能是否萎缩（无主力）
     3. 跌破任一则止损，不抱幻想
```

### 8.4 早盘应对策略输出格式

> 用户明确要求：持仓股次日早盘策略必须用以下格式输出

| 开盘情景 | 应对策略 |
|---------|---------|
| 高开（>+2%） | 具体操作 |
| 平开（-1%~+2%） | 具体操作 |
| 低开（-1%~-3%） | 具体操作 |
| 大低开（<-3%） | 具体操作 |

**每只持仓股必须输出**：
1. 关键价位（止损线 → 现价 → 成本价）
2. 集合竞价金额对比（判断主力意图）
3. 四个开盘情景的应对

## 九、市场陷阱清单（实战教训整合）

### 9.1 Friday Buying 陷阱（高频失败模式）

> 实盘记录反复出现：周五批量建仓 → 周一全部跑输大盘 → 亏损扩大

**正确做法：**
```
❌ 周五尾盘不开新仓（隔夜风险）
❌ 大盘涨时不追跟风股（买在日内高点）
❌ 不在板块轮动日追跟风股
✅ 等待周一开盘方向确认后再买
✅ 首次建仓不超过计划仓位的 50%
```

### 9.2 资金结构陷阱：超大单拉盘 + 小单出货 = 量化对倒

**诚邦股份(603316) 案例**：今日超大单净流入=+3599万，小单净流出=-998万
→ 超大单在拉盘，但小单在出货 = 量化资金在对手盘互相倒

**判断标准**：

| DDX/DDZ | 含义 | 操作 |
|---------|------|------|
| 超大单净流入 + 小单净流入 | 真实主力入场 | ✅ 关注 |
| 超大单净流入 + 小单净流出 | 量化对倒，可能是假拉升 | ⚠️ 谨慎 |
| 超大单净流出 + 小单净流入 | 主力撤退 | ❌ 排除 |
| 全线净流出 | 出货 | ❌ 排除 |

**教训**：不能仅看 DDX 正负，还要对比超大单 vs 小单的方向是否一致。不一致时以小单方向为准。

### 9.3 高开低走形态（亨通光电案例·2026-05-11）

**案例：亨通光电（600487）**
- 开盘79.30（+2.6%）→ 冲高79.90 → 单边下跌 → 收盘74.99（-2.95%）
- 收盘价 ≈ 日内最低价，典型的**光脚阴线收在最低点**看跌信号

**判断标准**：
```
✅ 看跌信号：收盘价 ≈ 日内最低价（阴线实体吃掉当日大部分涨幅）

```

**操作**：次日低开则止损，不格局。均线系统已转空（MA5<MA10<MA20），下跌趋势中不抄底。

### 连板检测算法（腾讯K线）

```python
def check_limit_ups(klines, days=10):
    """检测近N日涨停日，返回[(日期, 涨幅%)]"""
    result = []
    for i in range(max(1, len(klines)-days), len(klines)):
        prev = klines[i-1][2]
        chg = (klines[i][2] - prev) / prev * 100 if prev > 0 else 0
        if chg >= 9.7:
            result.append((klines[i][0], round(chg, 1)))
    return result

def check_consecutive_limit(klines, n=3):
    """检测最近是否出现连续n个交易日涨停"""
    closes = [k[2] for k in klines]
    for start in range(len(klines) - n):
        ok = all(
            (closes[start+i] - closes[start+i-1]) / closes[start+i-1] * 100 >= 9.7
            for i in range(1, n)
        )
        if ok:
            return True
    return False
```

## 十二、强制安全检查（⚠️ 选股/复盘必经步骤）

> 所有选股或复盘输出的观察股，在给出买卖建议前，**必须**完成以下两项安全检查。

### 12.1 杀猪盘扫描 — `trap-detector`

**触发时机**：每只候选股在进入买卖建议前，必须调用 `trap-detector` skill 进行 8 信号扫描。

**检查内容**：
1. 大量低质量账号同时推荐
2. 推荐话术模板化（"即将爆发"、"主力建仓完毕"、"目标翻倍"）
3. 付费社群 / VIP 直播间引流
4. 基本面与热度脱节
5. K 线异常配合（推荐密集期前已大幅拉升）
6. 老师 / 股神人设推广
7. 跨平台联动推广（小红书/抖音/B站/知乎）
8. 虚假研报 / 伪造消息

**评级标准**：

| 命中信号数 | 评级 | 操作 |
|-----------|------|------|
| 0-1 | 🟢 安全 | 正常进入后续分析 |
| 2-3 | 🟡 注意 | 进入，但需在建议中注明"有少量推广信号" |
| 4-5 | 🟠 警惕 | **排除，不给买入建议**，可加入自选跟踪 |
| 6+ | 🔴 高度可疑 | **直接排除**，强烈建议回避 |

**加权规则**：用户提到"朋友推荐/群里/老师带" → 严重程度+1；"内幕/稳赚" → +2；"必涨/翻倍" → +1。

### 12.2 龙虎榜席位分析 — `lhb-analyzer`

**触发时机**：候选股通过杀猪盘扫描（评级≤🟡）后，调用 `lhb-analyzer` skill 查询龙虎榜席位。

**检查内容**：
- 买席/卖席分布：机构 vs 游资 vs 量化席位
- **量化席位（拉萨系）排查**：买席中量化席位 ≥ 2 → 排除（次日程序化砸盘风险）
- 机构净买入 = 正向信号（真实主力）
- 沪股通/深股通净买入 = 正向信号（外资认可）
- 卖席量化席位集中 = 危险信号

**席位风险评级**：

| 席位特征 | 风险 | 操作 |
|---------|------|------|
| 无量化席位 + 机构净买 | 🟢 低 | 正常通过 |
| 无量化席位 + 无机构 | 🟡 中 | 可通过，仓位控≤10% |
| 量化席位 1个 | 🟠 中高 | 控仓≤5%，快进快出 |
| 量化席位 ≥ 2 | 🔴 高 | **排除**，不买 |
| 买席量化+卖席机构对倒 | 🔴 高 | **排除** |
