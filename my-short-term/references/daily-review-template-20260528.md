## RSI>95 连板股操作规则（2026-05-28 确认）

RSI>95的连板股判断：
- RSI=100不代表RSI>95，涨停导致的RSI=100是正常现象（溢出），不是排除依据
- RSI>95需是真实计算值（非涨停状态下的正常股票达到95+）
- 连板中 RSI>95 = 终极过热，次日必有大回调
- RSI从高位(80+)快速升至95+ = 主力拉升末端，1-2日内回撤

## 腾讯K线RSI批量查询代码模板

```python
import urllib.request, json, ssl

ctx = ssl.create_default_context()
ctx.check_hostname = False; ctx.verify_mode = ssl.CERT_NONE

def get_prefix(code):
    return 'sz' if code.startswith(('300','002','000','001')) else 'sh'

def calc_rsi(klines, n=6):
    closes = [float(k[2]) for k in klines[-30:] if k and len(k)>2]
    if len(closes) < 7: return None
    g, l = [], []
    for i in range(1, len(closes)):
        d = closes[i] - closes[i-1]
        g.append(max(d, 0)); l.append(max(-d, 0))
    ag = sum(g[-n:])/n; al = sum(l[-n:])/n
    return 100.0 if al==0 else round(100-100/(1+ag/al), 1)

# 批量查询
targets = [('002613','北玻股份'), ('002806','华锋股份'), ...]
for code, name in targets:
    prefix = get_prefix(code)
    url = f'https://web.ifzq.gtimg.cn/appstock/app/fqkline/get?_var=kline_dayhfq&param={prefix}{code},day,,,15,qfq'
    req = urllib.request.Request(url, headers={'User-Agent':'Mozilla/5.0'})
    with urllib.request.urlopen(req, timeout=8, context=ctx) as r:
        raw = r.read().decode('utf-8')
    data = json.loads(raw.split('=',1)[1])
    kd = data['data'][f'{prefix}{code}']
    klines = kd.get('qfqday') or kd.get('day') or []
    rsi = calc_rsi(klines)
    today_close = klines[-1][2] if klines else '-'
    print(f'{name} {code} {today_close} RSI={rsi}')
```
