# 腾讯K线数据陷阱（重要）

## 腾讯行情涨跌幅字段不可用

**⚠️ qt.gtimg.cn的涨跌幅字段（data[31]）对涨停股存在系统性失真，禁止用于判断真实涨跌。**

### 原因
腾讯行情涨跌幅字段受前复权/除权计算影响，收盘后数据与真实涨跌存在偏差。

### 正确做法

**方法1（推荐）：腾讯K线前复权收盘价计算**
```python
# 腾讯K线字段：k[0]=日期，k[1]=收盘，k[2]=开盘，k[3]=最高，k[4]=最低，k[5]=成交量(手)
klines = data['data'][p + code].get('qfqday') or data['data'][p + code].get('day', [])
last_close = float(klines[-1][1])
prev_close = float(klines[-2][1])
zhangfu = (last_close - prev_close) / prev_close * 100
# 成交额
vol_shou = float(klines[-1][5])  # 成交量(手)
amount_yi = vol_shou * 100 * last_close / 1e8
```
