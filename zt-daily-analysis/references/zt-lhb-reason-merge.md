# 涨停股完整列表（含上榜原因）

**2026-05-29 实测验证**

## 核心方法

akshare `stock_lhb_detail_em` 返回的上榜原因是标准格式化文本，可以直接匹配涨停池获取每只涨停股的驱动逻辑：

```python
import akshare as ak

# 步骤1：涨停池
zt = ak.stock_zt_pool_em(date='YYYYMMDD')
zt_codes = set(zt['代码'].astype(str).tolist())

# 步骤2：龙虎榜（含上榜原因）
lhb = ak.stock_lhb_detail_em(start_date='YYYYMMDD', end_date='YYYYMMDD')

# 步骤3：过滤出涨停股
lhb_zt = lhb[lhb['代码'].astype(str).isin(zt_codes)].sort_values('龙虎榜净买额', ascending=False)

for _, r in lhb_zt.iterrows():
    net_yi = r['龙虎榜净买额'] / 1e8
    reason = r['上榜原因']
    print(f"{r['名称']}({r['代码']}) 净买{net_yi:.2f}亿 原因:{reason}")
```

## 上榜原因分类

| 上榜原因关键词 | 含义 |
|--------------|------|
| 日涨幅偏离值达到7% | 单日10cm涨幅异常，需出龙虎榜 |
| 连续三个交易日内，涨幅偏离值累计达到20% | 连续3日异动（主板） |
| 日涨幅达到15% | 创业板/科创板20cm涨停 |
| 连续三个交易日内收盘价格涨幅偏离值累计达到30% | 连续3日涨30%（科创/创业板） |
| 日换手率达到20% | 高换手率上榜 |
| 日价格振幅达到15% | 日内振幅大 |

## 注意事项

1. **akshare `stock_lhb_detail_em` 无席位明细** — 返回列包含上榜原因但无 `OPERATEDEPT_NAME`，做席位分析需另用 MX接口 `client.query('{代码} 龙虎榜')`
2. **龙虎榜只覆盖上榜股票** — 未上榜的涨停股（如一字板）不在此结果中，需单独处理
3. **函数签名** — `start_date`/`end_date`（不是 `date`），格式 `YYYYMMDD`
