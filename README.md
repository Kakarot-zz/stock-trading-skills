# Stock Trading Skills

Hermes Agent 股票交易技能集，包含东方财富数据接口（dfcf）和A股短线交易策略（finance）两大类。

## 目录结构

```
dfcf/          — 东方财富妙想技能包
daily-review/ — 当日综合复盘
deep-analysis/— 个股深度分析
investor-panel/— 50位投资大佬评审团
lhb-daily-review/— 龙虎榜量化席位过滤
my-short-term/— 南瓜鱼短线选股体系
stock-launch-analysis/— K线形态选股
xiaomaolv/     — 小茅率短线选股
zhusceng-short-term/— 主升浪短线策略
zt-daily-analysis/— 涨停板结构分析
```

## dfcf 模块（东方财富）

- mx-data    — 行情、财务、关联关系数据查询
- mx-moni    — 模拟组合管理
- mx-search  — 资讯搜索
- mx-xuangu  — 条件选股
- mx-zixuan  — 自选股管理
