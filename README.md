A股超短线选股与风控体系，整合技术面+资金面+筹码面三大维度。

- v3.4.0 (2026-05-30)：新增第十一章「实战临盘体系」（5月行情提炼）
- v3.0.0~v3.3.0：整合徐总交易体系精华（趋势理论/量价口诀/板块轮动等）

**核心模块**：

| 章节 | 内容 |
|------|------|
| 第一~三章 | 技术面：RSI/MA/MACD/K线形态/缠论结构 |
| 第四~六章 | 资金面：涨停板结构/龙虎榜席位/量化席位识别 |
| 第七~九章 | 筹码面：股东人数/主力持仓/户均持股市值 |
| 第十章 | 基本面：财务指标/估值体系 |
| 第十一章 | 实战临盘体系（指数共振择时/选股流动性风控/仓位管理/风控自进化） |
| **第十二章** | **交易认知与运气管理体系（阿萨儿核心思想）** |

**第十二章新增四大技能**：

- **Skill 1.1** 历史镜像对标：找历史风格影子（2月长飞→5月688光纤复制）
- **Skill 1.2** 政策调子感知：hwj定调子→国产替代+688，锁定最强载体
- **Skill 2.1** 四节点复盘脚本：5.6/5.14/5.22/5.29四个关键节点操作原则
- **Skill 2.2** 震荡vs单边分级：单边市缩量防守，震荡市跟节点

- **Skill 3.2** 硬性熔断：15%红线+黑名单+共性复盘
- **Skill 4** 三维过滤矩阵：**节奏×风格×载体**，四步审判决策

**铁律**：
- 模拟账户仓位 ≤20% 单只
- -7% 硬止损
- 炸板次数 ≥10 的涨停股不买
- 量化席位 ≥2 个的标的跳过

### my-short-term — 南瓜鱼短线选股体系

基于"K线形态+资金意图"双重验证的选股策略。

**核心逻辑**：前期连续大涨 → 断板 → 大跌 → 缩量整理 → 尾盘低吸买入

**触发条件（需同时满足）**：
- 首板次日开盘涨幅在 -3%~0% 之间
- 分时图在 -1% 以下横盘整理，成交量极度萎缩
- 5日均线上方运行
- 前3日有涨停板（非当天）

**适用场景**：二连板后回调至5日线附近的买入机会

### daily-review — 当日综合复盘

资金面+逻辑面双层复盘框架。

**资金面**（7条策略驱动）：
1. 龙虎榜机构席位分析
2. 涨停板结构分析
3. 预期差逆势选股
4. 炸板率监测
5. 创业板情绪
6. 主流板块资金流
7. 全市场宽度

**逻辑面**：
- 板块节奏判断（存量博弈/正向运动）
- 三个明日加强信号（创业板接力/弱转强数量/逆向运动修复）
- 互动易用法
- 人气榜避坑
