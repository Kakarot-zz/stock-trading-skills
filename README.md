### my-short-term — 南瓜鱼短线选股体系

基于"K线形态+资金意图"双重验证的选股策略。

**核心逻辑**：前期连续大涨 → 断板 → 大跌 → 缩量整理 → 尾盘低吸买入

**触发条件（需同时满足）**：
- 首板次日开盘涨幅在 -3%~0% 之间
- 分时图在 -1% 以下横盘整理，成交量极度萎缩
- 5日均线上方运行
- 前3日有涨停板（非当天）

**适用场景**：二连板后回调至5日线附近的买入机会

### daily-review — 当日综合复盘

资金面+逻辑面双层复盘框架。

**资金面**（7条策略驱动）：
1. 龙虎榜机构席位分析
2. 涨停板结构分析
3. 预期差逆势选股
4. 炸板率监测
5. 创业板情绪
6. 主流板块资金流
7. 全市场宽度

**逻辑面**：
- 板块节奏判断（存量博弈/正向运动）
- 三个明日加强信号（创业板接力/弱转强数量/逆向运动修复）
- 互动易用法
- 人气榜避坑
