# 资金意图分析案例：派发 vs 主升浪（2026-05-22）

## 案例一：凯莱英(sz002821) — 拉高出货/派发模式

**走势特征**：近5日主力净流出-2.98亿，但股价从122→126反而上涨

| 日期 | 主力净流入 | 特征 |
|------|-----------|------|
| 05-14 | -79.5M | 净流出 |
| 05-15 | -123.7M | 最大流出日 |
| 05-18 | -29.6M | 净流出 |
| 05-19 | +7.1M | 微幅净流入 |
| 05-20 | -72.2M | 净流出 |

**量价结构**：
- 5/13~5/20：连续缩量阴跌（量比0.64~0.76x），股价从128→118
- 缩量不跌=主力控盘护盘（没出完货）
- 5/21：放量大阳+6%，但成交额仅0.12亿（无量拉升）

**判断**：主力在派发阶段，无量反弹是减仓机会。

## 两种模式的鉴别

| 特征 | 派发模式（凯莱英） | 主升浪模式（四环生物） |
|------|------------------|---------------------|
| 成交额 | 极低（0.12亿） | 持续放大（从0.3x→6.84x） |
| 上涨时量能 | 无量上涨 | 放量上涨 |
| 主力资金 | 连续净流出 | 净流入为主 |
| 横盘特征 | 缩量不跌（护盘出货） | 缩量横盘（筹码锁定） |
| 砸盘后 | 无反弹或反弹无量 | 快速缩量止跌再拉 |
| 板块 | 独立大盘，无板块联动 | 有板块或概念加持 |
